# 🛡️ Suresh Deora — Interactive Terminal Portfolio

An interactive, terminal-style portfolio for a cybersecurity professional.
Visitors *type commands* (or click them) to explore my profile — it boots
like a real system, with command history, tab-autocomplete and easter eggs.

🔗 **Live site:** https://sureshdeora.github.io  ← (add after deploy)

## ✨ Features
- Working command line: about, skills, projects, experience, education, certs, contact, resume
- `scan` runs a playful fake nmap; `sudo` is a surprise 🙃
- ↑ ↓ history, Tab autocomplete, Ctrl+L clear
- Clickable command chips for non-CLI visitors
- Boot sequence, matrix-rain backdrop, CRT scanlines
- Fully responsive · prefers-reduced-motion friendly · no build step

## 📂 Structure
- `index.html` — terminal shell
- `styles.css` — theme + CRT effect
- `app.js` — command engine + all content (edit the `DATA` object)
- `Suresh_Deora_Resume.pdf` — opened by the `resume` command

## 🚀 Run locally
    python3 -m http.server 8080   # then open http://localhost:8080
